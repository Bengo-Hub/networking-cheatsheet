# IPv6 Test Lab Address Allocation

This document provides sample IPv4 and IPv6 addresses for a virtual test lab environment, including IS-IS NET addresses.

---

## Address Block Summary

### IPv4 Allocation
- **Block:** `172.16.0.0/22`
- **Subnets:**
  - `172.16.0.0/24` - Router Loopback Addresses
  - `172.16.1.0/24` - Laptops/WiFi Addresses
  - `172.16.2.0/24` - Router Interface 1 Addresses
  - `172.16.3.0/24` - Router Interface 2 Addresses

### IPv6 Allocation
- **Block:** `2001:db8:1::/48`
- **Subnets:**
  - `2001:db8:1:0::/64` - Loopback Addresses
  - `2001:db8:1:1::/64` - Laptops Addresses
  - `2001:db8:1:2::/64` - Router Interface 1 Addresses
  - `2001:db8:1:3::/64` - Router Interface 2 Addresses

## Notes

- **Address Format**: IPv4 addresses use `/32` for loopback and host addresses, `/24` for subnets. IPv6 addresses use `/128` for loopback and `/64` for interface subnets.
- **NET Address Generation**: IS-IS NET addresses are derived from the IPv4 loopback address using the format: `49.0001.XXXX.XXXX.XXXX.00`
- **Test Lab Use**: These addresses are designed for virtual test lab environments and use the documentation prefix `2001:db8::/32` for IPv6.
